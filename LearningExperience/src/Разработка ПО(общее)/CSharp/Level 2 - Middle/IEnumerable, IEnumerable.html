Перебираемая коллекция должна реализовать интерфейс IEnumerable.

public interface IEnumerable : IEnumerable { // // Summary: // Returns an enumerator that iterates through the collection. // // Returns: // An enumerator that can be used to iterate through the collection. IEnumerator GetEnumerator(); } 109

IEnumerable is an interface that defines one method GetEnumerator which returns an IEnumerator interface, this in turn allows readonly access to a collection. A collection that implements IEnumerable can be used with a foreach statement.
