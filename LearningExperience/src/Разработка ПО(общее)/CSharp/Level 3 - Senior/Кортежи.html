Доступные с C# 7.0, кортежи представляют собой краткий синтаксис для группировки данных в легковесную структуру

(double, int) t1 = (4.5, 3);

Console.WriteLine($"Tuple with elements {t1.Item1} and {t1.Item2}.");

As the preceding example shows, to define a tuple type, you specify types of all its data members and, optionally, the field names. Beginning with C# 7.3, tuple types support equality operators == and !=.

(int a, byte b) left = (5, 10);

(long a, int b) right = (5, 10);

Console.WriteLine(left == right); // output: True

Console.WriteLine(left != right); // output: False
