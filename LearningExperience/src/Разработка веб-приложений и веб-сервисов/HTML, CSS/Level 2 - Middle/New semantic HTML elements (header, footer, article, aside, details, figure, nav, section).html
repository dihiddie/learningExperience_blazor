New Semantics

Семантический элемент четко описывает его значение как для браузера, так и для разработчика.

Примеры несемантических элементов: <div> и <span> - ничего не говорит о его содержимом.
Примеры семантических элементов: <form>, <table> и <article> - четко определяет его содержание.
Многие веб-сайты содержат HTML-код, например: <div id = "nav"> <div class = "header"> <div id = "footer"> для обозначения навигации, верхнего и нижнего колонтитула.

В HTML есть некоторые семантические элементы, которые можно использовать для определения различных частей веб-страницы:



<section>
    <h1>WWF</h1>
    <p>The World Wide Fund for Nature (WWF) is an international organization working on issues regarding the conservation.</p>
</section>

<article>
    <h2>Google Chrome</h2>
    <p>Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!</p>
</article>
элемент <aside> определяет некоторый контент помимо контента, в который он помещен (например, боковая панель).
