SqlConnection

Представляет подключение к базе данных SQL Server


private static void CreateCommand(string queryString,string connectionString)
{
using (SqlConnection connection = new SqlConnection(connectionString))
{
SqlCommand command = new SqlCommand(queryString, connection);
command.Connection.Open();
command.ExecuteNonQuery();
}
}

Объект SqlConnection представляет собой уникальный сеанс для SQL Server источника данных. В системе базы данных клиента или сервера она эквивалентна сетевому подключению к серверу. SqlConnection используется вместе с SqlDataAdapter и SqlCommand для повышения производительности при подключении к базе данных Microsoft SQL Server.
