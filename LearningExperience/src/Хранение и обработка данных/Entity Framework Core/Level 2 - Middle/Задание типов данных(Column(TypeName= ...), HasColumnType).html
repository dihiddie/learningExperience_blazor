• [Column(TypeName = "varchar(200)")]

    • Можно настроить столбцы, указав для столбца точный тип данных, так как при использовании реляционной бд поставщик выбирает тип данных, основанный на типе свойстве .net. Например, SQL Server сопоставляет DateTime свойства со datetime2(7) столбцами, а string свойства — nvarchar(max) со столбцами (или для nvarchar(450) свойств, которые используются в качестве ключа).
