Entity - класс, который мапится к таблице в базе данных. Этот класс должен быть включен как DbSet<TEntity> свойство типав классе DbContext. Api EF мапит каждую энтити к таблице и каждое свойство к колонкам таблицы.

Entity Data Model (EDM) - грубо говоря, схема взаимоотношений энтити (слепок со схемы БД). Выбираем для создания новый айтем в проекте с типом ADO.NET Entity Data Model, устанавливаем коннекшен до бд, задаем., генерировать названия классов в единственном или во множественном числе и используем ли внешние ключи и вуаля! Схема готова.
