dotnet CLI

Если вдруг ва не хочется тащить идею весом в несколько гигабайт, отличная программа для работы в командной строке — .NET Core CLI. Основной компонент .NET Core CLI называется dotnet. Его можно использовать для решения практически любых задач разработки, в частности, для создания, сборки, тестирования и упаковки проектов.

Команда dotnet new делает то же самое, что элемент меню File – New Project в Visual Studio. С её помощью можно создавать проекты различных типов.

  • new

    • Используйте команду dotnet new, чтобы вывести список предустановленных шаблонов.

    run

    Чтобы собрать и запустить приложение, используется команда dotnet run.

  • build

    • Строет проект и все его зависимости.
