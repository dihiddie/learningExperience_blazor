Управление зависимостями с использованием csproj или CLI

<PackageReference>
<PackageReference Include="PACKAGE_ID" Version="PACKAGE_VERSION" />

Атрибут Include указывает идентификатор пакета, который нужно добавить в проект. Атрибут Version указывает версию, которую нужно получить. Версии указываются в соответствии с правилами версии NuGet.

Чтобы добавить новую зависимость или удалить имеющуюся:

  • csproj

    • Чтобы добавить/удалить зависимость, добавьте элемент <PackageReference> внутри элемента <ItemGroup>. Вы можете добавить к существующей <ItemGroup> или создать новую.

  • CLI

    • Чтобы добавить зависимость, выполните команду dotnet add package

    dotnet add package Microsoft.EntityFrameworkCore
