Fluent API -механизм, наряду с аннотациями данных, который помогает добавить доп правила кофнигурации, чтобы переопределить используемые условности. Представляет собой набор методов, которые определяют сопоставление между классами и их св-ми, таблицами и их столбцами.

Настраивать можно:

  • конфигурация EF модели для маппинга в бд
  • конфигурация энтити к таблице и маппинг отношений, т.е. главные и внешние ключи, индексы, названия таблиц, один-к-одному, и тд
  • конфигурация свойств - маппинг колонок, например, назвние колонки, дефолтное значение, nullable
