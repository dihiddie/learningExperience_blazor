Начиная с Entity Framework 6, каждый раз, когда Entity Framework отправляет команду в базу данных, эта команда может быть перехвачена кодом приложения. Это чаще всего используется для логирования SQL, но также может использоваться для изменения или отмены команды.

Настраивается все это великолепие через context.Database.Log = Console.Write;

Логируется много всякого, вот самое интересное:

  • Sql для разных типов команд
  • Параметры
  • Выполняется ли асинхронно
  • Датавремя начала выполнения команды
  • Результат выполнения команды
