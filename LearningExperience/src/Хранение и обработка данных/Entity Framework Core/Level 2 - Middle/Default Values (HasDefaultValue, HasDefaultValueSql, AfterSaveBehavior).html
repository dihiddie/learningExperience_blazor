• HasDefaultValue - указывает дефолтное значение колонки при добавлении в бд, если оно не задано явно
  • HasDefaultValueSql - указывает функцию, которая будет использоватья для генерирования значения колонки при добавлении в бд, если оно не задано явно
  • AfterSaveBehaivor - если указано, задает, что делать со свойством после сохранения:
    • Ignore - любые изменения будут игнорироваться
    • Save - все сохраняется в базу как обычно
    • Throw - если задано новое значение или изменено старое, будет выброшено исключение
