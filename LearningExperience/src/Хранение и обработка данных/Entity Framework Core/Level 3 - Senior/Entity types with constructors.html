Начиная с EF Core 2.1, теперь можно определить конструктор с параметрами и заставить EF Core вызывать этот конструктор при создании экземпляра сущности.

Когда EF Core создает экземпляры этих типов, например для результатов запроса, он сначала вызывает конструктор без параметров по умолчанию, а затем устанавливает для каждого свойства значение из базы данных. Однако, если EF Core находит параметризованный конструктор с именами параметров и типами, соответствующими сопоставленным свойствам, он вместо этого вызывает параметризованный конструктор со значениями для этих свойств и не устанавливает каждое свойство явно.
