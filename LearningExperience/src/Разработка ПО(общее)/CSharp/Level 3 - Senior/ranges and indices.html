В C# 8.0 добавлены две новые структуры Index и Range по работе с индексами и диапазонами.

Получить второй элемент Index idx = 2; array[idx]
Получить три последних элемента array[^3]
Получить значения со 2 по 5 Range range= 2..5; array[range]
Получить массив значения со второго элемента с конца по конец массива array[^2..^0] | array[^2..] | Range range= ^2..^0; array[range]
Получить первые три элемента массива array[..3];
