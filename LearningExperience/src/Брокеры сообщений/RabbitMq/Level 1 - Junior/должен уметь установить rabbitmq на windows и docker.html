Docker

Устанавливаем докер на локальную машину

https://hub.docker.com/editions/community/docker-ce-desktop-windows/

Скачиваем контейнер:

docker pull rabbitmq:3-management

Запускаем контейнер

# docker run -d --hostname localhost -p 15672:15672 -p 5672:5672 -e RABBITMQ_DEFAULT_USER=mqadmin -e RABBITMQ_DEFAULT_PASS=mqadminpassword rabbitmq:3-management
