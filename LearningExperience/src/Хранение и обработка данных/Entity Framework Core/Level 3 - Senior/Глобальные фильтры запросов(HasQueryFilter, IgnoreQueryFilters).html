Метод HasQueryFilter Fluent API Entity Framework Core используется для применения глобального фильтра запроса к определенной сущности, чтобы фильтр был включен во все операторы запроса, созданные для сущности EF Core. Наиболее распространенные варианты использования этой функции включают свойства, указывающие, что экземпляр объекта удален или не активен иным образом, а также идентификатор клиента в приложении, в котором хранятся данные для нескольких клиентов.

protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        modelBuilder.Entity<Magazine>().HasQueryFilter(m => EF.Property<int>(m, "MagazineId") == 1);
    }

Если вы не хотите применять фильтр, используйте метод IgnoreQueryFilters
