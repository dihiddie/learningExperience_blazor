RPC (запрос / ответ) - популярный шаблон для реализации с брокером обмена сообщениями, таким как RabbitMQ. Типичный способ сделать это - для клиентов RPC отправлять запросы, которые направляются в долгоживущую (известную) очередь сервера. Сервер (ы) RPC обрабатывает запросы из этой очереди, а затем отправляет ответы каждому клиенту, используя очередь, указанную клиентом в заголовке ответа.

Источник: https://www.rabbitmq.com/direct-reply-to.html, https://www.rabbitmq.com/tutorials/tutorial-six-dotnet.html
