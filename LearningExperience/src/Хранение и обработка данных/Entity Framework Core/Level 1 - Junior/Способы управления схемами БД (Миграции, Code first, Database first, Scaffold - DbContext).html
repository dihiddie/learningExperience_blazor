EF Core не поддерживает визуальный конструктор для модели БД и мастера для создания классов сущностей и контекста, подобных EF 6. Итак, нам нужно выполнить обратное проектирование с помощью команды Scaffold-DbContext. Эта команда обратного проектирования создает классы сущностей и контекста (путем получения DbContext) на основе схемы существующей базы данных.
