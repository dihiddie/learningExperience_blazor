Чтобы визуализировать свое приложение на высших уровнях, его логическую архитектуру, можно создать диаграмму зависимостей в Visual Studio. Чтобы убедиться, что ваш код остается правильным по дизайну, провалидируйте его с диаграммой зависимостей

Диаграмма позволяет вам организовать солюшн в логические абстрактные группы, называемые слои. Их можно использовать, что описать основные задачи или компоненты. Каждый слой может содержать другие слои для лучшего описания. Зависимости, обозначенные стрелками, показывают взаимосвязь между слоями. Чтобы поддерживать архитектуру кода, установите желаемые зависимости на диаграмме, а потом провалидируйте код по ней
