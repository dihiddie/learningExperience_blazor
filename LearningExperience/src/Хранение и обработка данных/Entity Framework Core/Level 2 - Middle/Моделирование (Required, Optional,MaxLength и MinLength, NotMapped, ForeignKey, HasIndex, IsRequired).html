• Required - указывает, что свойство не может быть null
  • IsRequired - версия Fluent APi для Required
  • optional - свойство считается таковым, если оно принимает null
  • MaxLength - применяется к свойству, чтобы указать максимальное количество символов или байтов для столбца, которому должно соответствовать свойство.
  • MinLength - применяется к свойству, чтобы указать минимальное количество символов или байтов для столбца, которому должно соответствовать свойство.
  • NotMapped - указывает, что энтити или столбец не должны быть смаплены в бд
  • HasIndex - используется для создания индекса базы данных по столбцу, сопоставленному с указанным свойством сущности.
