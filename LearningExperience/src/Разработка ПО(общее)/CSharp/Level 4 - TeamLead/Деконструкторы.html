Деконструкторы (не путать с деструкторами) позволяют выполнить декомпозицию объекта на отдельные части. Кортежи поддерживают ее изначально,

Чтобы тип поддерживал деконструкцию, нужно релаизовать следующий метод:

public void Deconstruct(out Type1 var1, ..., out TypeN varN) { … }

public void Deconstruct(out string name, out int age)

{

name = this.Name; age = this.Age;

}

Использование:

Person person = new Person { Name = "Tom", Age = 33 };

(string name, int age) = person;

Console.WriteLine(name); // Tom

Console.WriteLine(age); // 33
