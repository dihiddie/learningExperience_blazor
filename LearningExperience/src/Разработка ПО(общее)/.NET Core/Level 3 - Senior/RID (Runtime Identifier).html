RID - это сокращение от Runtime Identifier. Значения RID используются для идентификации целевых платформ, на которых выполняется приложение. Они используются пакетами .NET для представления ресурсов конкретной платформы в пакетах NuGet. Они вступают в игру, когда пакеты содержат ресурсы, зависящие от платформы (например, собственный код для Linux или 64-битной Windows).
