Fuslogvw.exe (средство просмотра журнала привязки сборок)

Если случилась ошибка “Could not load file or assembly” или какая то схожая с ней, то вам понадобится больше информации о сборке, которая породила эту проблему. Для этого:

  • Включить Enable Fusion Logs (Assembly Binding Logging). Либо через регистр, либо через саму программу
  • Вызвать проблему
  • Смотреть все привязки для вашего приложения
