• ValueGeneratedOnAdd - указывает, что бд сама заполняет это свойство при добавлении новой записи. EF Core игнорирует это свойство при создании инструкции INSERT
  • ValueGeneratedNever - указывает, что для данного свойство не учитывается значение, которое генерирует бд. Полезно, если хотите обойти какую-то дефолтную логику бд
  • ValueGeneratedOnAddOrUpdate - указывает, что бд сама заполняет это свойство при добавлении новой записи или при изменении существующей. Поэтому свойство не следует включать в операторы INSERT или UPDATE, когда SQL создается EF Core
