BindRequired и BindNever

BindRequired - указывает, что свойство обязательно должно быть установлено

BindNever - указывает, что свойство не должно быть установлено напрямую никогда

Если для свойства с атрибутом BindRequired не будет передано значение, то в объект ModelState будет помещена информация об ошибках, а свойство ModelState.IsValid возвратит false. И в данном случае, проверяя значение ModelState.IsValid, мы можем проверить корректность создания объекта User.

FromHeader

данные берутся из заголовков запроса


    public void OnGet([FromHeader(Name = "Accept-Language")] string language)
FromQuery

данные берутся из строки запроса

FromRoute

данные берутся из значений маршрута


    [HttpGet("{facilityId}", Name = "GetTotalBandwidth")]
    public IActionResult GetTotalBandwidth([FromRoute] int facilityId)


    Complete route definition: /api/Settings/GetTotalBandwidth/{facilityId}
    Complete received url: /api/Settings/GetTotalBandwidth/100
FromForm

данные берутся из полученных форм. Атрибут FromForm предназначен для входящих данных из представленной формы, отправленной приложением типа контента / x-www-url-formencoded.

FromBody

Чтобы заставить web-api прочитать значимый тип из реквест-боди, нужно добавить FromBody атрибут в параметр

public HttpResponseMessage Post([FromBody] string name) { ... }

В этом примере веб-API будет использовать media-type formatter для чтения значения name из тела запроса.
