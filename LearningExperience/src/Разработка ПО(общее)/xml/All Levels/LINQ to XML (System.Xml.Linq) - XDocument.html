Когда использовать XMLDocument, а когда XDocument?

Если вы используете .NET версии 3.0 или меньше, вы должны использовать XmlDocument aka классический DOM API. А если есть выбор, стоит использовать XDocument aka LINQ to XML. С ним намного проще создавать документы и работать с ними.

  • Намного компактнее, чем XmlDocument

    • XDocument doc = new XDocument( new XElement("root", new XAttribute("name", "value"), new XElement("child", "text node")));

  • Поддердка неймспейсов и аттрибутов

    • XNamespace ns = "http://somewhere.com"; XElement element = new XElement(ns + "elementName");
