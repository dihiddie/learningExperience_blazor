Конкретного класса ApiController больше нет, поскольку MVC и WebAPI были объединены в ASP.NET Core. Но класс Controller MVC предоставляет набор функций, которые вам, вероятно, не понадобятся при разработке только веб-API, таких как представления и привязка модели.

Используйте класс ControllerBase в пакете Microsoft.AspNetCore.Mvc.Core.
