Query Types должен быть настроен в методе OnModelCreating, как показано ниже. Обратите внимание, что в отличие от типа объекта, вам потребуется класс DbQuery для включения типов запросов в DbContext. Метод ToView используется для указания источника данных (имя таблицы или имя представления) для типа запроса.


protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.Query<AuthorBook>().ToView("AuthorBookCount");
}
