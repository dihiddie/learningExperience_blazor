CoreCLR

Эта среда CLR построена на той же кодовой базе, что и среда CLR. Первоначально CoreCLR был средой выполнения Silverlight и был разработан для работы на нескольких платформах, в частности Windows и OS X. CoreCLR теперь является частью .NET Core и представляет собой упрощенную версию CLR. Это все еще кроссплатформенная среда выполнения, теперь включающая поддержку многих дистрибутивов Linux. CoreCLR также является виртуальной машиной с возможностями JIT и выполнения кода.

Установка

Два пути, как установить SDK на свою машину:

  • Использовать установшик
    • Инсталяторы в первую очередь предназначены для машин разработчиков. SDK распространяется с использованием собственного механизма установки каждой поддерживаемой платформы, например пакетов DEB в Ubuntu или пакетов MSI в Windows. Эти установщики устанавливают и настраивают среду так, чтобы пользователь мог использовать SDK сразу после установки. Однако они также требуют прав администратора на машине. По умолчанию SDK устанавливается «бок о бок» (SxS), что означает, что несколько версий могут одновременно существовать на одном компьютере в любой момент времени.
  • Использовать скрипты установки

    • Если использовать скрипты, то для этого не требуются права администратора. Однако они также не устанавливают на машину никаких дополнительных пред-установок; вам необходимо установить все необходимые компоненты вручнуюю.
