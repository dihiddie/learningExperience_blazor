Visual Studio SDK помогает расширять возможности Visual Studio или интегрировать новые функции в Visual Studio. Вы можете распространять свои расширения для других пользователей, а также для Visual Studio Marketplace. Ниже приведены некоторые способы расширения Visual Studio:

  • Добавление команды, кнопок, меню и другие элементы пользовательского интерфейса в IDE
  • Добавить окна инструментов для новой функциональности
  • Расширить IntelliSense для данного языка или предоставить IntelliSense для новых языков программирования
  • Используйте лампочки для предоставления советов и предложений, которые помогут разработчикам написать лучший код
  • Включить поддержку нового языка
  • Добавить пользовательский тип проекта
