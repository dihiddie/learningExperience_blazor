Хранилище пакетов среды выполнения

Начиная с .NET Core 2.0, можно упаковывать и развертывать приложения для известного набора пакетов, существующих в целевой среде. Преимущества заключаются в более быстром развертывании, меньшем использовании дискового пространства и в некоторых случаях улучшенной производительности запуска. Эта функция реализована в виде хранилища пакетов среды выполнения, которое представляет собой каталог на диске, где хранятся пакеты. В этом каталоге есть подкаталоги для архитектур и целевых платформ. Структура файла аналогична расположению ресурсов NuGet на диске
