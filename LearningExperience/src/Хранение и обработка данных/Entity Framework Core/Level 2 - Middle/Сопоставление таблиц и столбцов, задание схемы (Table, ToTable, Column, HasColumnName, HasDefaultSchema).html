• Table("") - указывает название таблицы в бд, к которой мапится энтити
  • ToTable - Fluent Api версия Table атрибута
  • Column("") - указывает название столбца, к которому должно мапится свойство, если названия не совпадают
  • HasColumnName - Fluent Api версия Column атрибута
  • HasDefaultSchema - указывает схему в бд, в которой будут создаваться объекты, если она не настроена явно
